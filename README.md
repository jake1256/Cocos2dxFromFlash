Cocos2dxFromFlash
=================

Flashからcocos2dxに変換するツールです。

対応Ver
読み込み
Flash CS5〜

書き出し
cocos2d-x v2.x
cocos2d-x v3.x

複数ある画面サイズに対応するため、
top , center , bottom各種相対ポジション対応
各種Animation出力対応

例えば画面レイアウト製作をした時にnewアイコンだけアニメーションさせたい
といった使用方法も可能。

もちろんキャラクターのアニメーションのような複雑な組み合わせ、
腕の動きのような中心点がズレたアニメーションでも出力可能です。

開発はbitbucketのプライベートリポジトリで行っています。
